# Настройка позиционирования элементов шапки

Этот файл описывает CSS-переменные, которые используются в `static/css/style.css` для точной подстройки расположения и размеров элементов шапки (логотип, заголовок, поле поиска, переключатель языка, корзина).

Файл стилей использует переменные в секции `:root`. Изменяйте значения переменных — изменения применяются после перезагрузки страницы (при необходимости сбросьте кеш браузера Ctrl+F5).

## Как использовать
- Откройте `static/css/style.css` и найдите секцию `:root` в начале файла.
- Поменяйте значение переменной и сохраните файл.
- Обновите страницу в браузере.

## Таблица переменных

- **`--container-max-width`** — максимальная ширина контейнера страницы.
  - По умолчанию: `1200px`
  - Влияет на: ширину блоков `.container`.
  - Пример: `--container-max-width: 1400px;`

- **`--container-padding`** — внутренний отступ слева/справа для `.container`.
  - По умолчанию: `24px`

- **Размеры хедера / логотипа / заголовка**
  - **`--hero-inner-height`** — высота внутреннего хедера (по умолчанию `70px`).
  - **`--hero-inner-height-lg`**, **`--hero-inner-height-sm`** — альтернативные высоты для медиазапросов.
  - **`--hero-logo-height`**, **`--hero-logo-height-lg`**, **`--hero-logo-height-sm`** — высота логотипа в разных точках.
  - **`--hero-title-size`**, **`--hero-title-size-lg`**, **`--hero-title-size-sm`** — размер текста заголовка.
  - Влияют на: визуальную высоту шапки и размеры логотипа / заголовка.

- **Поле поиска**
  - **`--search-width-vw`** — относительная ширина поля поиска (пример: `68vw`).
  - **`--search-max-width`** — максимальная ширина поля поиска (пример: `760px`).
  - **`--search-height`** — высота поля поиска (пример: `44px`).
  - Влияют на: ширину и высоту инпута `header-search`.
  - Пример увеличения ширины: `--search-width-vw: 80vw; --search-max-width: 900px;`

- **Оффсеты для точной подстройки (по пикселям)**
  Для каждого блока доступны переменные `--<block>-top/right/bottom/left`.

  1) Логотип / левый блок шапки
     - `--hero-langs-top` — вертикальное смещение от верхней границы (по умолчанию `1px`).
     - `--hero-langs-left` — смещение слева (по умолчанию `12px`).
     - Пример: поднять логотип на 5px: `--hero-langs-top: -5px;`

  2) Заголовок
     - `--hero-title-top`, `--hero-title-left`, `--hero-title-right`, `--hero-title-bottom` — двигайте заголовок относительно его текущей позиции.
     - Пример: опустить заголовок на 6px: `--hero-title-top: 6px;`

  3) Поле поиска
     - `--header-search-top`, `--header-search-left`, `--header-search-right`, `--header-search-bottom` — смещают сам блок `.header-search`.
     - Пример: поднять поиск на 10px: `--header-search-top: -10px;`
      - `--search-padding-vert` — вертикальные отступы внутри поля поиска (по умолчанию `10px`).
      - `--search-padding-horiz` — горизонтальные отступы внутри поля поиска (по умолчанию `18px`).
      - `--search-border-radius` — радиус скругления поля поиска (по умолчанию `28px`).
      - `--search-font-size` — размер шрифта в поле поиска (по умолчанию `1rem`).
 
  4) Переключатель языков
      - `--hero-lang-switch-top`, `--hero-lang-switch-right`, `--hero-lang-switch-left`, `--hero-lang-switch-bottom`.
      - `--lang-btn-gap` — расстояние между кнопками переключения языков (RU / UA), по умолчанию `8px`.
      - `--lang-btn-padding` — внутренние отступы кнопок переключения языков (вертик. и гориз. значения), по умолчанию `6px 8px`.
      - `--lang-btn-font-size` — размер текста в кнопках переключателя (по умолчанию `0.9rem`).
      - `--lang-btn-border-radius` — радиус скругления кнопок переключения языка (по умолчанию `8px`).
     - Пример: сдвинуть вправо на 20px: `--hero-lang-switch-right: 72px;`

  5) Корзина
     - `--hero-cart-top`, `--hero-cart-right`, `--hero-cart-left`, `--hero-cart-bottom`.
     - Пример: поднять корзину на 4px и отступ справа 8px:
       ```css
       --hero-cart-top: 4px;
       --hero-cart-right: 8px;
       ```
    - `--cart-bubble-size` — диаметр кружка с количеством товаров в корзине (по умолчанию `22px`).

## Примеры быстрых правок
- Шире поле поиска, но не больше 900px:
```css
:root{
  --search-width-vw: 80vw;
  --search-max-width: 900px;
}
```

- Подвинуть переключатель языка немного левее (чтобы не налезал на корзину):
```css
:root{
  --hero-lang-switch-right: 110px;
}
```

- Опустить заголовок на мобильных устройствах (добавьте в медиазапрос или измените переменную для `--hero-title-size-sm`):
```css
@media (max-width:560px){
  :root{ --hero-title-top: 6px; }
}
```

## Отдельные настройки для полной и мобильной версии

Вы можете задавать отдельные значения переменных для полной (desktop) и мобильной версии двумя способами:

- Изменять значения прямо в корневом `:root` для глобальных (desktop) настроек.
- Переопределять те же переменные внутри медиазапроса `@media (max-width:560px)` для мобильных значений.

Пример: глобальная (desktop) настройка и переопределение для мобильной версии:
```css
:root{
  /* desktop */
  --search-width-vw: 68vw;
  --search-max-width: 760px;
  --search-height: 44px;
  --lang-btn-gap: 8px;
}

@media (max-width:560px){
  :root{
    /* mobile overrides */
    --search-width-vw: 94vw;
    --search-max-width: 520px;
    --search-height: 38px;
    --lang-btn-gap: 4px;
  }
}
```

Совет: если хотите быстро протестировать мобильные значения в браузере на десктопе, откройте DevTools, переключитесь на mobile device toolbar и измените переменные в инспекторе стилей — это позволяет подобрать нужные числа без повторного сохранения файлов.

## Советы
- Используйте маленькие шаги: меняйте значение на 1–5px и смотрите результат.
- После правки CSS обновляйте страницу с очисткой кеша (Ctrl+F5), чтобы увидеть изменения сразу.
- Если хотите — могу добавить файл `design-tune.md` с набором рекомендуемых значений для desktop/tablet/mobile.
